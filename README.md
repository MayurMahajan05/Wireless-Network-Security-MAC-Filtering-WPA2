# Wireless Network Security Project

## Secure WiFi Network using WPA2 and MAC Filtering

---

## 📌 Project Description
This project demonstrates the setup of a secure wireless network using Cisco Packet Tracer. The network includes both wired and wireless devices with security mechanisms like WPA2 authentication and MAC address filtering to prevent unauthorized access.

---

## 🛠️ Technologies Used
- Cisco Packet Tracer
- Wireless Networking
- WPA2 Security
- MAC Address Filtering
- DHCP

---

## 🌐 Network Overview
- SSID: MySecureWiFi
- Router IP: 192.168.0.1
- DHCP Enabled

---

## 🔐 Security Features
- WPA2 Password Protection
- MAC Address Filtering (Only authorized devices allowed)
